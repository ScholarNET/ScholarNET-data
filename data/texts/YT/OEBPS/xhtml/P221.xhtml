وَهَذَا اللَّفْظ عذب وَلَكِن لَيْسَ فِيهِ جَمِيع مَا فِي بَيت المتنبي وَقَوله
(أَنْت الْجواد بِلَا من وَلَا كدر ... وَلَا مطال وَلَا وعد وَلَا مذل) // من الْبَسِيط //
وَقَوله
(بِي حر شوق إِلَى ترشفها ... ينْفَصل الصَّبْر حِين يتَّصل)
(فالثغر وَالْفَجْر والمخلخل والمعصم ... دائي والفاحم الرجل) // من المنسرح //
وَقَوله
(وَلَكِن بالفسطاط بحرا أزرته ... حَياتِي ونصحي والهوى والقوافيا) // من الطَّوِيل //
وَقَوله
(أَمينا وإخلافا وعذرا وخسة ... وجبنا أشخصا لحت لي أم مخازيا) // من الطَّوِيل //
وَمِنْهَا إرْسَال الْمثل فِي أَنْصَاف الأبيات

كَقَوْلِه
(مصائب قوم عِنْد قوم فَوَائِد ... ) // من الطَّوِيل //
وَقَوله
(وَمن قصد الْبَحْر اسْتَقل السواقيا ... ) // من الطَّوِيل //
وَقَوله
(وَخير جليس فِي الزَّمَان كتاب) // من الطَّوِيل //
