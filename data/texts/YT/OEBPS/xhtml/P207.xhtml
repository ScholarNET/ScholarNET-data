وَقَوله
(يُسمى الحسام وَلَيْسَت من مشابهة ... وَكَيف يشْتَبه المخدوم والخدم)
(كل السيوف إِذا طَال الضراب بهَا ... يَمَسهَا غير سيف الدولة السأم) // من الْبَسِيط //
وَقَوله
(تهاب سيوف الْهِنْد وَهِي حدائد ... فَكيف إِذا كَانَت نزارية عربا) // من الطَّوِيل //
وَقَوله
(تحير فِي سيف ربيعَة أَصله ... وطابعه الرَّحْمَن وَالْمجد صاقل) // من الطَّوِيل //
وَقَوله
(قلد الله دولة سيفها أَنْت ... حساما بالمكرمات محلى)
(فَإِذا اهتز للندى كَانَ بحرا ... وَإِذا اهتز للعدا كَانَ نصلا) // من الْخَفِيف //
وَقَوله
(وَأَنت حسام الْملك وَالله ضَارب ... وَأَنت لِوَاء الدّين وَالله عَاقد) // من الطَّوِيل //
وَقَوله
(لقد سل سيف الدولة الْمجد معلما ... فَلَا الْمجد مخفيه وَلَا الضَّرْب ثالمه)
(على عاتق الْملك الْأَغَر نجاده ... وَفِي يَد جَبَّار السَّمَوَات قائمه)
(وَإِن الَّذِي سمى عليا لمنصف ... وَإِن الَّذِي سَمَّاهُ سَيْفا لظالمه)
(وَمَا كل سيف يقطع الْهَام حَده ... وتقطع لزبات الزَّمَان مكارمه) // من الطَّوِيل //
