(يَا أَيهَا السَّائِل عَن مذهبي ... ليقتدي فِيهِ بمنهاجي)
(منهاجي الْعدْل وقمع الْهوى ... فَهَل لمناجي من هاجي) // من السَّرِيع //
وَله
(يَقُولُونَ ذكر الْمَرْء يحيا بنسله ... وَلَيْسَ لَهُ ذكر إِذا لم يكن نسل)
(فَقلت لَهُم نسلي بَدَائِع حكمتي ... فَإِن فاتنا نسل فَإنَّا بهَا نسلو) // من الطَّوِيل //
وَله
(نَصَحْتُك جامل الإخوان طرا ... على عذب سقوه أَو أجاج)
(وَلَا ترج الصفاء بِغَيْر مذق ... فَلَا يَخْلُو السراج من السناج) // من الوافر //
وَله
(إِذا مَا هَمَمْت بكشف الظُّلم ... وَحفظ الثغور وسد الثلم)
(فعول على خلتين اثْنَتَيْنِ ... خرق الحسام ورفق الْقَلَم) // من المتقارب //
وَله
(لَا يعْدم الْمَرْء كُنَّا يستكن بِهِ ... ومنعة بَين أهليه وَأَصْحَابه)
(وَمن نأى عَنْهُم قلت مهابته ... كالليث يحقر إِمَّا غَابَ عَن غابه) // من الْبَسِيط //
