وَقَالَ
(قَالَ رُؤْيا الْمَنَام عنْدك حق ... قلت هَيْهَات كل ذَاك بخار)
(لَيْت يقظانهم يَصح لَهُ الْأَمر ... فَكيف المغط والنخار) // الْخَفِيف //
وَقَالَ
(سَرِير بت بماخور ... على دف وطنبور)
(وَصَوت الطبل كردم طع ... وَصَوت الناي طلير)
(فصرنا من حمى الْبَيْت ... كأنا وسط تنور)
(وصرنا من أَذَى الصفع ... كَمثل الْعَمى والعور)
(لقد أَصبَحت مخمورا ... وَلَكِن أَي مخمور) // الهزج //
وَقَالَ من قصيدة
(ترى العقيان كالذهب الْمُصَفّى ... تركب فَوق أثفار الدَّوَابّ)
(وكيسي مِنْهُ خلو مثل كفي ... أما هَذَا من الْعجب العجاب) // الوافر //
وَقَالَ
(قَامَ للشقوة أيري ... وَجرى بالنحس طيري)
(وَولى حل سراويلك ... يَا مولَايَ غَيْرِي)
(وتقرأت علينا ... كسعيد بن جُبَير)
(أَتَرَى قد عقر النَّاقة ... يَا مولَايَ أيري)
(لَيْسَ لي مِنْك سوى صبحك ... الله بِخَير) // مجزوء الرمل //
