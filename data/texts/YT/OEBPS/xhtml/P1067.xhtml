(نبهت مِنْهُ لحاجتي عمرا ... وَلم أعول فِيهَا على عَمْرو) // المنسرح //
يُرِيد قَول بشار
(إِذا أيقظتك حروب العدا ... فنبه لَهَا عمرا ثمَّ نم) // المتقارب //
وَللْآخر
(المستجير بِعَمْرو عِنْد كربته ... كالمستجير من الرمضاء بالنَّار) // الْبَسِيط //
وَقَالَ
(عذرت الْأسد أَن صليت بناري ... مخاطرة فَمَا بَال الْكلاب)
(وَأَزْوَاج الْحَرَائِر لم يجابوا ... لدي فَكيف أَزوَاج القحاب) // الوافر //
وَقَالَ وَقد قَالَ لَهُ بعض الرؤساء مَا أشبهك فِي الإبرام إِلَّا بِابْن أبي رَافع
(ضربت فِي الإبرام يَا سَيِّدي ... لي مثلا بِابْن أبي رَافع)
(فَقلت فِي ذَلِك لَا تعجبوا ... من متخم يفسو على جَائِع) // السَّرِيع //
وَقَالَ
(إِنِّي بليت بِأَقْوَام مواعدهم ... تزيدني فَوق مَا أَلْقَاهُ من محن)
(وَمن يذقْ لسعة الأفعى وَإِن سلمت ... مِنْهَا حشاشته يفزع من الرسن) // الْبَسِيط //
