صَاحبهَا الْمُسَمّى كوشيار مِنْهَا
(وَكنت سَمَاء والعجاج سحائبا ... وخيلك أبراجا وجيشك أنجما)
(وأنزلت مِنْهَا كوشيار وَإِنَّمَا ... تقنصت من فَوق المجرة ضيغما)
(عرفتك صياد الْأسود وَلم أكن ... عرفتك صياد الْأسود من السما)
(خدمتكم يَا آل بوية مُدَّة ... غَدا بَينهَا فرخ الْوَسَائِل قشعما) // من الطَّوِيل //
وَمن أُخْرَى فِي أبي الْحُسَيْن الْمُزنِيّ
(كلم هِيَ الْأَمْثَال إِلَّا أَنَّهَا ... فِي النَّاس قد أضحت بِلَا أَمْثَال)
(فَإِذا لقين فَإِنَّهُنَّ عوالي ... وَإِذا شممن فَإِنَّهُم غوالي) // من الْكَامِل //
وَمن صاحبية
(تَأَخّر عَن كتبي الْجَواب وَإِنَّمَا ... تَأَخّر برد المَاء عَن كبد حرى)
(فَلَا تفسدن عشْرين ألفا وهبتها ... بِعشْرين حرفا كلامك تستمرى) // من الطَّوِيل //
وَمن ميكالية
(فديتك مَا بدا لي قصد حر ... سواك من الورى إِلَّا بدا لي)
(وَإنَّك مِنْهُم وكذاك أَيْضا ... من المَاء الفرائد واللآلي)
(وتسكن دَارهم وَكَذَلِكَ سُكْنى الْحِجَارَة ... والزمرد فِي الْجبَال) // من الوافر //
