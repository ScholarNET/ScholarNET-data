ملح من شعره فِي الهجاء والمجون

قَالَ فِي ابْن متويه
(يَا فَتى متوي رفقا ... لست من يُنكر أَصله)
(إِنَّمَا يُنكر مِنْهُ ... من جُنُون فِيهِ ثقله)
(أَنْت نذل من كرام ... أَنْت فِي الطاووس رجله) // مجزوء الرمل //
كَأَنَّهُ مقلوب بَيت المتنبي
(فَإِن تفق الْأَنَام وَأَنت مِنْهُم ... فَإِن الْمسك بعض دم الغزال) // الوافر //
وَقَالَ فِي مَعْنَاهُ
(أَبوك أَبُو عَليّ ذُو عَلَاء ... إِذا عد الْكِرَام وَأَنت نجله)
(وَإِن أَبَاك إِذْ تغزى إِلَيْهِ ... لكالطاووس يقبح من رجله) // الوافر //
وَقَالَ فِيهِ
(أَحْمد هَذَا سبط متويه ... فِي مَوته بعد غَد تهنيه)
(والشأن فِي أَنِّي على بغضه ... أحتاج أَن أقعد للتعزيه) // السَّرِيع //
وَقَالَ فِيهِ
(قَالَ ابْن متويه لأَصْحَابه ... وَقد حشوه بأيور العبيد)
(لَئِن شكرتم لأزيدنكم ... وَإِن كَفرْتُمْ فعذابي شَدِيد) // السَّرِيع //
وَقَالَ فِيهِ
(أَبْصرت فِي كف ابْن متوي عَصا ... فَسَأَلته عَنْهَا ليوضح عذرا)
