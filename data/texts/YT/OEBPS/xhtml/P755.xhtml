(اذا دنت بَيْضَاء مَكْرُوهَة ... مني نأت بَيْضَاء محبوبه) // السَّرِيع //
وَقَوله من مجزوء الْكَامِل
(قَالُوا تكهل من هويت ... فَقلت رسم قد دثر)
(عَايَنت من طلابه ... زمرا مُوَاصلَة زمر)
(وكذاك اصحاب الحَدِيث ... نفاقهم عِنْد الْكبر) // مجزوء الْكَامِل //
وَقَوله من المتقارب
(بَكَيْت بدمع يفوق السَّحَاب ... الى ان جرى المَاء حَولي وساحا)
(وَلَو لم اكن رجلا سابحا ... غرقت والزمت نَفسِي الجناحا) // المتقارب //
وَقَوله من الْبَسِيط
(ليل المحبين مطوي جوانبه ... مشمر الذيل مَنْسُوب الى الْقصر)
(مَا ذَاك الا لَان الصُّبْح نم بِنَا ... فَأطلع الشَّمْس من غيظ على الْقَمَر) // الْبَسِيط //
وَقَوله من مجزوء الوافر
(بَدَائِع خَدّه ورد ... صوالج صُدْغه سبج)
(اذا اتَّصَلت محاسنه ... نقطع بَينهَا المهج) // مجزوء الوافر //
وَقَوله وَهُوَ مِمَّا يسْتَغْفر مِنْهُ من الْبَسِيط
(يَا قَاسم الرزق لم خانتني الْقسم ... مَا انت مُتَّهم قل لي من اتهمَ)
(ان كَانَ نجمي نحسا انت خالقه ... فَأَنت فِي الْحَالَتَيْنِ الْخصم وَالْحكم) // الْبَسِيط //
وَقَوله فِي امرد التحى من السَّرِيع
(انْظُر الى ميت وَلكنه ... خلو من الاكفان والغاسل)
