(فَلَمَّا نهضت أَتَانِي الْكتاب ... وَجَاء الْهَدَايَا ووافى الرَّسُول)
(وَقَالَت تَقول بِنَا يَا فَتى ... فَقلت وأنعظت لم لَا أَقُول) // المتقارب //

(وَأجر غلماني فِي وَاسِط ... جوع وَكَانُوا لَا يرامونا)
(جادوا بِمَا كنت ضنينا بِهِ ... فاتسعوا مِمَّا يناكونا)
(لَو أَن رِزْقِي مثل أدبارهم ... كنت من الإثراء قارونا) // السَّرِيع //

(فَإِنَّهَا القفل مَوْضُوع على خربه) // الْبَسِيط //
