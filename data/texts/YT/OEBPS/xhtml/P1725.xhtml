وَمن أُخْرَى
(مَضَت الشبيبة والحبيبة فَالتقى ... دمعان فِي الأجفان يزدحمان)
(مَا أنصفتني الحادثات رمينني ... بمودعين وَلَيْسَ لي قلبان) // من الْكَامِل //
وَمن أُخْرَى
(قلت للعين حِين شامت جمالا ... فِي وُجُوه كواذب الإيماض)
(لَا تغرنك هَذِه الْأَوْجه الغر ... فيا رب حَيَّة من رياض) // من الْخَفِيف //
وَمن أُخْرَى
(عذيري من ضحك غَدا سَبَب البكا ... وَمن جنَّة قد أوقعت فِي جَهَنَّم)
(لِأَنَّك لَا تروين بَيْتا لشاعر ... سوى بَيت من لم يظلم النَّاس يظلم) // من الطَّوِيل //
وَمن أُخْرَى
(عذيري من تِلْكَ الْوُجُوه الَّتِي غَدَتْ ... مناظرها للناظرين معاركا)
(عذيري من تِلْكَ الجسوم الَّتِي غَدَتْ ... سبائك تفنى النَّاس فِيهَا السبائكا) // من الطَّوِيل //
وَمن أُخْرَى
(خليلي عهدي بالليالي صوافيا ... فَمَا بالها أبدلن جيما بصادها)
(خليلي هَل أبصرتما مثل أدمعي ... نفدن وَحقّ الله قبل نفادها) // من الطَّوِيل //
وَمن أُخْرَى
(يفل غَدا جَيش النَّوَى عَسْكَر اللقا ... فرأيك فِي سح الدُّمُوع موفقا)
(وَخذ حجتي فِي ترك جَنْبي سالما ... وقلبي وَمن حقيهما أَن يشققا)
