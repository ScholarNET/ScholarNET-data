(ليل حكى الْغرْبَان سُودًا لَونه ... وَكَأن أنجمه البزاة الْبيض) // من الْكَامِل //
وَقَالَ
(يَكْفِيك أَن الْهوى لم يبْق فِي جَسَدِي ... من الْجَوَارِح عضوا غير مَجْرُوح)
(إِنِّي نحلت الْهوى قلبِي فأنحلني ... حَتَّى غَدا جَسَدِي أخْفى من الرّوح) // من الْبَسِيط //
وَقَالَ
(نَفسِي فدَاء غزال مَا اكتحلت بِهِ ... إِلَّا تصورته أنموذج الْحور)
(وَكلما رام نطقا وَهُوَ مبتسم ... فالدر مَا بَين منظوم ومنثور)
(أضحى جنى النَّحْل ممزوجا بريقته ... لكنما الخصر مِنْهُ خصر زنبور) // من الْبَسِيط //
وَقَالَ
(أرى الْفطر عيد النَّاس فِي كل بَلْدَة ... ووجهك لي عيد ورؤيته فطري)
(إِذا مَا أعد النَّاس للفطر عطرهم ... فحسبي بِمَا فِي عَارِضَيْك من الْعطر) // من الطَّوِيل //
وَقَالَ
(قُم إِلَى الراح فاسقنيها فَفِيهَا ... قُوَّة للفتى وقرة عين)
(مَا ترى الصَّوْم صَار بالأسودين ... وأتانا شَوَّال بالأحمرين) // من الْخَفِيف //
وَقَالَ
(صديقك قد ألم بِهِ صديق ... وأعوزه الشَّرَاب الأرجواني)
(وَقد بعثا إِلَيْك وَلَيْسَ شَيْئا ... سوى مَعْهُود فضلك يرجوان) // من الوافر //
وَقَالَ
(لَا تعرضن على الروَاة قصيدة ... مَا لم تبالغ قبل فِي تهذيبها)
