نبذ من ذكر سرقاته

من ذَلِك قَوْله
(شيخ فَتى والشباب أَكْثَرهم ... قد علم الله غير فتيَان) // المنسرح //
من قَول كثير
(يَا عز هَل لَك فِي شيخ فَتى أبدا ... وَقد يكون شباب غير فتيَان) // الْبَسِيط //
وَقَوله
(وَأَوْلَاد الْحَرَائِر لم يجابوا ... لدي فَكيف أَوْلَاد القحاب) // الوافر //
من قَول دعبل
(إِنِّي لأهجو من يجود بِمَالِه ... أتظنني أدع اللَّئِيم الواضعا)
(على أَنِّي أَظُنك سَوف تنجو ... بعرضك من يَدي منجى الذئاب) // الْكَامِل //
من قَول أبي الزيات
(نجا بك لؤمك منجى الذئاب ... حمته مقاذره أَن ينالا) // المتقارب //
وَقَوله
(وَأحسن مَا رَأينَا قطّ رَاحا ... إِذا كَانَت مَطِيَّة كأس رَاح) // الوافر //
من قَول أبي تَمام
(رَاح إِذا الراح كن مطيها ... كَانَت مطايا الشوق فِي الأحشاء) // الْكَامِل //
وَقَوله
(سترت بظله من ريب دهري ... فعز على النوائب أَن تراني) // الوافر //
