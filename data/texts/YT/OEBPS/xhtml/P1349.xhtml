وَقَوله فِي غَيره
(رغيفك فِي الْأَمْن يَا سَيِّدي ... يحل مَحل حمام الْحرم)
(فَللَّه دَرك من سيد ... حرَام الرَّغِيف حَلَال الْحرم) // المتقارب //
وَقَالَ من أَبْيَات
(يَعْلُو ويعلى وكل من سجيته ... يَعْلُو الكنيف ويعلى بالغراميل) // الْبَسِيط //
وَقَالَ فِي رجل ارْتَفع قدره وَكَانَ أَبوهُ حلاجا
(أَقُول وَقد قَالُوا ابْن مأسدة غَدا ... على مركب لَا من حمير أَبِيه)
(وَلَا الصَّوْت محلاج وَلَا السرج لوحه ... وَلَا حب قطن كالشعير بِفِيهِ)
(مقَال الْوَلِيد البحتري فَإِنَّهُ ... قد أَنبأَنَا عَن مثله وَذَوِيهِ)
(مَتى أرت الدُّنْيَا نباهة خامل ... فَلَا ترتقب إِلَّا خمول نبيه) // الطَّوِيل //
وَقَالَ فِي قينة
(لنا قينة تَحْمِي من الشّرْب شربنا ... فقد أمنُوا سكرا وَخَوف خمار)
(تكشر عَن أنيابها فِي غنائها ... فتحكي حمارا شم بَوْل حمَار) // الطَّوِيل //
وَقَالَ فِي شَاعِر
(مَا قَالَ بَيْتا مرّة ... وَلَا يَقُول مَا بَقِي)
(وكل شعر قَالَه ... فإثمه فِي عنقِي) // مجزوء الرجز //
