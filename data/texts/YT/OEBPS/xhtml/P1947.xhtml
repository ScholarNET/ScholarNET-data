(فتكت بمهجتي عمدا فَهَلا ... طويت الجرم فِي ثني اعتذارك)
(أرى نَار الصدود على فُؤَادِي ... فَمَا بَال الدُّخان على عذارك) // من الوافر //
وَقَالَ
(بنفسي من نَفسِي لَدَيْهِ رهينة ... يجرعها صبرا ويمنعها الصبرا)
(أغار على قلبِي فَلَمَّا استباحه ... أغار على دمعي فنظمه ثغرا) // من الطَّوِيل //
وَقَالَ
(وقائلة مَا بَال خدك كلما ... رَآنِي يلقاني بصفرة جِلْبَاب)
(فَقلت كَذَا بدر السَّمَاء إِذا بدا ... أَفَاضَ على الغبراء صفرَة زرياب) // من الطَّوِيل //
وَقَالَ
(عجبت لفاقع سحنتي ومدامعي ... منهلة ورأته قبل موردا)
(فأجبتها لَا تعجبن فَإِنَّهُ ... يصفر لون الزَّعْفَرَان من الندا) // من الْكَامِل //
وَقَالَ
(يَا ذَا الَّذِي فتن الورى وبوجهه ... أخيا رسوما للمحاسن عافيه)
(يَحْكِي محياه خلال عذاره ... علم السَّلامَة فِي طراز العافيه) // من الْكَامِل //
وَقَالَ
(إِذا رمت من سيد حَاجَة ... فراع لَدَيْهِ الرِّضَا وَالْغَضَب)
(فَإِن التهجم ليل المنى ... وَإِن الطلاقة صبح الْأَدَب) // من المتقارب //
وَقَالَ
(لَا تحسبن هشاشتي لَك عَن رضى ... فوحق فضلك إِنَّنِي أتملق)
