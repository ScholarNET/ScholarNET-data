وَسمعت أَبَا الْفَتْح عَليّ بن مُحَمَّد البستي يَقُول أنشدت لأبي عِيسَى
(رغيف أبي عَليّ حل خوفًا ... من الْأَسْنَان ميدان السماك)
(إِذا كسروا رغيف أبي عَليّ ... بَكَى يبكي فيهو باكي) // الوافر //
فبنيت عَلَيْهِ قولي لبَعض من أطايبه
(لنا شيخ بفقحته يواسي ... ويحلق شاربيه بالمواسي)
(إِذا بايته فِي جَوف بَيت ... فسا يفسو فسَاء فَهُوَ فاسي) // الوافر //
وَلأبي عِيسَى
(لوم النديم منغص ... طيب الْمجَالِس والندام)
(وسماحة الْحر الْكَرِيم تزيد ... فِي طيب المدام)
(فَإِذا شربت الراح فاشربها مَعَ النَّفر الْكِرَام)
(وتنكبن مَا اسطعت ... أَخْلَاق اللئام بني اللئام) // مجزوء الْكَامِل //
وَلأبي الْفَتْح بن المنجم
(كنت أَدْعُو عَلَيْهِ بالشعر حَتَّى ... زَاده الشّعْر فِي الْأَنَام جمالا)
(وَإِذا كَانَ هَكَذَا كَانَ خذلاني ... دَقِيقًا وَكَانَ شؤمي جلالا)
