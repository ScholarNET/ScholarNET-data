وَقَوله
(خير الطُّيُور على الْقُصُور وشرها ... يأوي الخراب ويسكن الناووسا) // من الْكَامِل //
وَقَوله
(لَيْسَ الْجمال لوجه صَحَّ مارنه ... أنف الْعَزِيز بِقطع الْعِزّ يجتدع) // من الْبَسِيط //
وَقَوله
(وَلَيْسَ يَصح فِي الأفهام شَيْء ... إِذا احْتَاجَ النَّهَار إِلَى دَلِيل) // من الوافر //
قَالَ ابْن جني هَذَا كَمَا يَقُول أهل الجدل من شكّ فِي المشاهدات فَلَيْسَ بكامل الْعقل
وَقَوله
(وَقد يتزيا بالهوى غير أَهله ... ويستصحب الْإِنْسَان من لَا يلائمه) // من الطَّوِيل //
وَقَوله
(وَمَا تَنْفَع الْخَيل الْكِرَام وَلَا القنا ... إِذا لم يكن فَوق الْكِرَام كرام) // من الطَّوِيل //
