(وعاث فِي سفرتي كمشبلة ... غرثى بِتِلْكَ الأنامل اللبقه)
(قلعا وبلعا بِلَا مراقبة ... لله فِي عيلتي وَلَا شفقه)
(قل للرئيس الَّذِي أنامله ... مبسوطة بالنوال منخرقه)
(حلت لي الْميتَة الَّتِي حرمت ... فَكيف تنبو نَفسِي عَن الصدقه) // المنسرح //
وَقَالَ
(يَا سيدا ظلّ فَردا فِي سيادته ... يخْشَى ويرجى لدفع الْحَادِث الجلل)
(الشوق ينهضني والعدم يقعدني ... فَمن شناك بِهِ مَا بِي من الْخلَل) // الْبَسِيط //
وَقَالَ
(جملَة أَمْرِي أنني مُفلس ... وَلَيْسَ للْمُفلس إخْوَان)
(وكل ذِي عَيْش بِلَا دِرْهَم ... فعيشه ظلم وعدوان) // السَّرِيع //
وَقَالَ
(قيل مَا أَعدَدْت للبرد ... فقد جَاءَ بشده)
(قلت دراعة عري ... تحتهَا جُبَّة رعده) // مجزوء الرمل //
وَقَالَ
(وجاهل قَالَ لي لَا بُد من فرج ... فَقلت للغيظ لم لَا بُد من فرج)
(فَقَالَ من بعد حِين قلت يَا عجبا ... من يضمن الْعُمر لي يَا بَارِد الْحجَج)
(وَلَو كَانَ مَا قلت حَقًا لم أكن رجلا ... مقسم الْعُمر فِي الروحات والدلج)
