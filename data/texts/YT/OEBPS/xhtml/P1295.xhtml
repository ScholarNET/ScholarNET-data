(أَهْوى لتقبيل يَدي ... فَقلت لَا بل شفتي) // مجزوء الرجز //
وَقَالَ
(قل لأبي الْقَاسِم إِن جِئْته ... هنيت مَا أَعْطَيْت هنيته)
(كل جمال فائق رائق ... أَنْت برغم الْبَدْر أُوتِيتهُ) // السَّرِيع //
وَقَالَ
(قل لأبي الْقَاسِم الْحُسَيْنِي ... يَا نَار قلبِي وَنور عَيْني)
(الْبَدْر زين السَّمَاء حسنا ... وَأَنت زين لكل زين) // مخلع الْبَسِيط //
وَقَالَ من بَاب الإقتباس من الحَدِيث
(ومهفهف يُغني عَن الْقَمَر ... قمر الْفُؤَاد بفاتن النّظر)
(خالسته تفاح وجنته ... من غير إبْقَاء وَلَا حذر)
(فأخافني قوم فَقلت لَهُم ... لَا قطع فِي ثَمَر وَلَا كثر) // الْكَامِل //
وَقَالَ فِي مثله
(قَالَ لي إِن رقبي ... سيء الْخلق فداره)
(قلت دَعْنِي وَجهك ... الْجنَّة حفت بالمكاره) // مجزوء الرمل //
وَقَالَ فِي مثله
(أَقُول وَقد رَأَيْت لَهُ سحابا ... من الهجران مقبلة إِلَيْنَا)
(وَقد سحت غزالتها بهطل ... حوالينا الصدود وَلَا علينا) // الوافر //
