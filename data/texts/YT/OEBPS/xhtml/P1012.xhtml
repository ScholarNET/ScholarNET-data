(وَكلما عبتموه عِنْدِي ... زَاد جنوني بِهِ وعجبي) // مخلع الْبَسِيط //
وَقَالَ
(أَحْبَبْت بَدْرًا مَا لَهُ مشبه ... فِي الْحسن لَوْلَا أَنه جافي)
(أحور فِي مقلته حجَّة ... للعين والشين مَعَ الْقَاف)
(وَفِي ارتجاج الردف دَاع إِلَى ... نون وياء قبل مَا كَاف)
(سَأَلته الْوَصْل فَلم يحتشم ... وَقَالَ قدم نقدك الوافي) // السَّرِيع //
وَقَالَ
(يَا سَيِّدي ومؤملي ... قد شفني شوقي إليكا)
(دمعي عَلَيْك مورد ... فَكَأَنَّهُ من وجنتيكا) // مجزوء الْكَامِل //
وَقَالَ فِي غُلَام أعرج
(قَالُوا بليت بأعرج فأجبتهم ... الْعَيْب يحدث فِي غصون البان)
(مَاذَا عَليّ إِذا استجدت شمائلا ... وروادفا تغني عَن الكثبان)
(إِنِّي أحب جُلُوسه وأريده ... للنوم لَا للجري فِي الميدان)
(فِي كل عُضْو مِنْهُ حسن كَامِل ... مَا ضرني أَن زلت القدمان) // الْكَامِل //
(لَيْسَ شرب المدام للمستهام ... مذهبا مَا بِهِ من الأسقام)
(كلما دبت المدامة فِي الْأَعْضَاء ... دب اشتياقه فِي الْعِظَام) // الْخَفِيف //
وَقَالَ فِي غُلَام رش عَلَيْهِ مَاء الْورْد
(لَيْت شعري عَن مَاء وردك هَذَا ... هُوَ من وجنتيك أم شفتيكا)
