وَقَوله
(تمتّع من سهاد أَو رقاد ... وَلَا تَأمل كرى تَحت الرجام)
(فَإِن لثالث الْحَالين معنى ... سوى معنى انتباهك والمنام) // من الوافر //
قَالَ ابْن جني أَرْجُو أَن لَا يكون أَرَادَ بذلك أَن نومَة الْقَبْر لَا انتباه لَهَا
وَمِنْهَا استكراه التَّخَلُّص

قَالَ القَاضِي لَعَلَّك لَا تَجِد فِي شعره تخلصا مستكرها إِلَّا قَوْله
(أحبك أَو يَقُولُوا جر نمل ... ثبيرا وَابْن إِبْرَاهِيم ريعا) // من الوافر //
فَأَما قَوْله
(فأفنى وَمَا أفنته نَفسِي كَأَنَّمَا ... أَبُو الْفرج القَاضِي لَهُ دونهَا كَهْف) // من الطَّوِيل //
وَقَوله
(لَو اسْتَطَعْت ركبت النَّاس كلهم ... إِلَى سعيد بن عبد الله بعرانا) // من الْبَسِيط //
وَقَوله
(أعز مَكَان فِي الدنا سرج سابح ... وَخير جليس فِي الزَّمَان كتاب)
(وبحر أَبُو الْمسك الخضم الَّذِي لَهُ ... على كل بَحر زخرة وعباب) // من الطَّوِيل //
فَهِيَ وَإِن لم تكن مستحسنة مختارة فَلَيْسَتْ بالمستهجن السَّاقِط
