وَأَخذه أَبُو الطّيب فَلم يحسن فِي تَكْرِير لفظ النهب وَذكر القماش إِذْ هُوَ من أَلْفَاظ الْعَامَّة
(وَنهب نفوس أهل النهب أولى ... بِأَهْل الْمجد من نهب القماش) // من الوافر //
3 - وَقَالَ بشار بن برد
(كَأَن مثار النَّقْع فَوق رؤوسنا ... وأسيافنا ليل تهاوى كواكبه) // من الطَّوِيل //
أَخذه أَبُو الطّيب وَذكر الرماح مَكَان الأسياف فَقَالَ
(وكأنما كسي النَّهَار بهَا دجى ... ليل وأطلعت الرماح كواكبا) // من الْكَامِل //
4 - وَقَالَ مُسلم بن الْوَلِيد
(أَرَادوا ليخفوا قَبره من عدوه ... فطيب تُرَاب الْقَبْر دلّ على الْقَبْر) // من الطَّوِيل //
ألم بِهِ أَبُو الطّيب فَقَالَ
(وَمَا ريح الرياض لَهَا وَلَكِن ... كساها دفنهم فِي الترب طيبا) // من الوافر //
5 - وَقَالَ الفرزدق
(وَكنت فيهم كممطور ببلدته ... يسر أَن جمع الأوطان والمطرا) // من الْبَسِيط //
أَخذه أَبُو الطّيب فَقَالَ
(وَلَيْسَ الَّذِي يتبع الوبل رائدا ... كمن جَاءَهُ فِي دَاره رائد الوبل) // من الطَّوِيل //
6 - وَفِي قَوْله فِي هَذِه القصيدة
(وخيل إِذا مرت بوحش وروضة ... أَبَت رعيها إِلَّا ومرجلنا يغلي) // من الطَّوِيل //
