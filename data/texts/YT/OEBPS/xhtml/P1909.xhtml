(فأريته من عبرتي ... مَا سَالَ من دَمه نجيعا) // من مجزوء الْكَامِل //
وَقَالَ
(وغزال منحته خَالص الود ... فجازى بالصد والاجتناب)
(لم ألمه أَن أتقى بحجاب ... ردني واله الْفُؤَاد لما بِي)
(هُوَ روحي وَلَيْسَ يُنكر للروح ... توار عَن الورى بالحجاب) // من الْخَفِيف //
وَله
(كتبت إِلَيْهِ أستهدي وصالا ... فعللني بوعد فِي الْجَواب)
(أَلا لَيْت الْجَواب يكون خيرا ... فيشفي مَا أحَاط من الجوى بِي) // من الوافر //
وَقَالَ
(ظَبْي يحار الْبَرْق فِي بريقه ... غنيت عَن إبريقه بريقه)
(فَلم أزل أرشف من رحيقه ... حَتَّى شفيت الْقلب من حريقه) // من الرجز //
وَقَالَ
(شافه كفي رشا ... بقبلة مَا شفت)
(فَقلت إِذْ قبلهَا ... يَا لَيْت كفي شفتي) // من مجزوء الرجز //
وَقَالَ
(من لي كَفِيلا بشمل الْأنس أجمعه ... بشادن حل فِيهِ الْأنس أجمعه)
(مَا زَالَ يعرض عَن وَصلي فأخدعه ... فَالْآن لي لَان بعد الصد أخدعه) // من الْبَسِيط //
