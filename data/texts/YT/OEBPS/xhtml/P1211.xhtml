(بل ببلوغ الْمَنِيّ وقاصية البغية ... عفوا وَنُهْبَة الأمل) // المنسرح //
وَقَالَ فِي الْمُغنِي الْقرشِي
(إِذا غناني الْقرشِي يَوْمًا ... وعناني بِرُؤْيَتِهِ وضربه)
(وددت لَو أَن أُذُنِي مثل عَيْني ... هُنَاكَ وَأَن عَيْني مثل قلبه) // الوافر //
وللمهلبي فِي هَذَا الْمَعْنى
(إِذا غناني الْقرشِي ... دَعَوْت الله بالطرش)
(وَإِن أَبْصرت طلعته ... فوا لهفي على العمش) // مجزوء الوافر // وَقَالَ مِنْهُ أَيْضا
(إِذا غنى لنا أمما ... حشوت مسامعي صمما)
(وَإِن أَبْصرت طلعته ... كحلت نواظري بعمى) // مجزوء الوافر //
