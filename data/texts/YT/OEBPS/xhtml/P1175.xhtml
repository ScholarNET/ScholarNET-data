وَقَالَ وأجاد
(ستعلمون مَا يكون مني ... إِن مد من ضبعي طول سني)
(أأدع الدُّنْيَا وَلم تدعني ... وسعت أيامي وَلم تسعني)
(أفضل عَنْهَا وتضيق عني ... ) // الرجز //
وَقَالَ من أُخْرَى
(تجاذبني يَد الْأَيَّام نَفسِي ... ويوشك أَن يكون لَهَا الغلاب)
(نهضت وَقد قعدن بِي اللَّيَالِي ... فَلَا خيل أعز وَلَا ركاب)
(وَمَا ذَنبي إِذا اتّفقت خطوب ... مغاضبة وَأَيَّام غضاب)
(وَبَعض الْعَدَم مأثرة وفخر ... وَبَعض المَال منقصة وَعَابَ)
(بناني والعنان إِذا نبت بِي ... رَبِّي أَرض ورجلي والركاب)
(سَوَاء من أقل الترب منا ... وَمن وارى معالمه التُّرَاب) // الوافر //
كَأَنَّهُ من قَول ابْن نباتة
(وَمن لبس التُّرَاب كمن علاهُ ... )
رَجَعَ
(وَإِن مزايل الْعَيْش اختصارا ... مساوي للَّذين بقوا فشابوا)
(وأولنا العناء إِذا طلعنا ... إِلَى الدُّنْيَا وآخرنا الذّهاب)
(وَإِن مقَام مثلي فِي الأعادي ... مقَام الْبَدْر تنبحه الْكلاب)
(رموني بالعيوب ملفقات ... وَقد علمُوا بِأَنِّي لَا أعاب)
