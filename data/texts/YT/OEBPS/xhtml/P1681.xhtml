(على رَأسهَا نجل لَهَا لم تجنه ... حشاها وَلَا علته قطّ لبانا)
(يشرد فِي أَعْلَاهُ كل دجنة ... يشق جلابيب الظلام سِنَانًا) // من الطَّوِيل //
وَقَالَ فِي الْكُرْسِيّ
(ومقعد لي وطئ ... يقوم عِنْد قعودي)
(يزهي بصدر فسيح ... رحب وبأس شَدِيد)
(لَهُ رواق أَدِيم ... على سواري حَدِيد)
(إِذا جَلَست عَلَيْهِ ... خلت الْأَنَام عَبِيدِي) // من المجتث //
وَفِيه أَيْضا
(ومرتبة من بوادي الْمُلُوك ... بَين الْقيام وَبَين الْقعُود)
(تمد بساطا لمستوطئ ... ثُبُوته عمد من حَدِيد) // من المتقارب //
وَفِيه أَيْضا
(ومستوقف لجُلُوس الْحُضُور ... على أَربع فِي الثرى موثقه)
(يمد على فَرعه مفرشا ... وَيظْهر فِي خصره مَنْطِقه)
(فَمن شَاءَ صيره مقْعدا ... وَمن شَاءَ صيره مرفقه) // من المتقارب //
وَقَالَ فِي طست الشمع
(وحديقة تهتز فِيهَا دوحة ... لم ينمها ترب وَلَا أمطار)
(فصعيدها صفر ونامي غصنها ... شمع وَمَا قد أثمرته نَار) // من الْكَامِل //
