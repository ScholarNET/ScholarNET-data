وَقَالَ هَل تعرفُون نظيرا لمعناها فِي شعر الْمُحدثين فَقلت لَا أعرف إِلَّا قَول البحتري
(وَمن عجب الدَّهْر أَن ... الْأَمِير أصبح أكتب من كَاتبه) // المتقارب //
فَقَالَ جودت وأحسنت وَهَكَذَا فَلْيَكُن الْحِفْظ وَقَالَ
(عزمت على الفصد يَا سَيِّدي ... لفضل دم كظني مؤلم)
(فَلَمَّا تَأَخَّرت عَن مجلسي ... أرقت لغير افتصاد دمي) // المتقارب //
وَقَالَ
(ومهفهف شكل المجون ... أضنى فُؤَادِي بالفتون)
(فنسيمه ملْء الأنوف ... وَحسنه ملْء الْعُيُون) // مجزوء الْكَامِل //
وَقَالَ
(فَمن كَانَ يقطف ورد الْجنان ... فقطفي مذ كنت ورد الخدود)
(وهمي مذ كنت در الثغور ... إِذا اهتم غَيْرِي بدر الْعُقُود) // المتقارب //
وَقَالَ
(كُنَّا وَأَسْبَاب الْهوى متفقه ... نبتا من الْورْد مَعًا فِي ورقه)
(فَالْآن إِذْ أَسبَابه مفترقه ... قد صَارَت الأَرْض علينا حلقه) // الرجز //
وَقَالَ
(يَا خاطرا يخْطر فِي تيهه ... ذكرك مَوْقُوف على خاطري)
(إِن لم تكن آثر من ناظري ... عِنْدِي فَلَا متعت بالناظر) // السَّرِيع //
