(فترى لحيته فِي استي ... إِلَى الصدغ كَمَا هِيَ) // مجزوء الرمل //
وَقَالَ من الوافر
(وشعري سخفه لَا بُد مِنْهُ ... فقد طبنا وَزَالَ الإحتشام)
(وَهل دَار تكون بِلَا كنيف ... فَيمكن عَاقِلا فِيهَا الْمقَام) // الوافر //
وَقَالَ
(تراني سَاكِنا حَانُوت عطر ... فَإِن أنشدت ثار لَك الكنيف) // الوافر //
وَقَالَ
(شعري الَّذِي أَصبَحت فِيهِ ... فضيحة بَين الملا)
(لَا يستجيب لخاطري ... إِلَّا إِذا دخل الخلا) // مجزوء الْكَامِل //
وَمن أُخْرَى
(أَلا أَيهَا الْأُسْتَاذ دَعْوَة شَاعِر ... طَرِيقَته فِي الشّعْر لَا تتبهرج)
(إِذا أَنْت وظفت القوافي فَخَيرهَا ... وَإِن قل مَا يَرْجُو وَمَا يتروج)
(وَمن كَانَ يحوي الْعطر دكان شعره ... فَإِنِّي كناس وشعري لَهُ مخرج) // الطَّوِيل //
وَقَالَ من قصيدة فِي بعض الوزراء خَالِيَة من السخف
(وهذي القصيدة مثل الْعَرُوس ... موشحة بالمعاني الملاح)
(بِلَا نفحة من فسا عَارض ... وَلَا وزن خردلة من سلَاح)
(فَلَو أَنَّهَا جعلت خطْبَة ... لكَانَتْ تحل عُقُود النِّكَاح)
(بعثت بهَا عنبرا فِي الشتَاء ... وَفِي الصَّيف كافور خرط رياحي)
