وَمن أُخْرَى
(سَيِّدي مَا أَظُنهُ ... بعد يدْرِي بِمَا جرى)
(مَا درى أَن عَبده ... فلسه قد تقشرا)
(عِنْد قوم معروفهم ... فِي قد صَار مُنْكرا)
(كنت كالمسك مرّة ... بِالدَّنَانِيرِ أَشْتَرِي)
(فَأَنا الْيَوْم بعد مَا ... صرت شَيخا كَمَا ترى)
(عبد من عِنْده نَبِيذ ... إِذا كَانَ أحمرا)
(خمرة دنها يضمن ... مسكا وعنبرا)
(كم فَم ذاقها فطاب ... وَقد كَانَ أبخرا)
(وَغُلَام بكأسها ... رَاح يسْعَى وبكرا)
(هُوَ فِينَا بريحها ... عبق قد تعطرا)
(ظلّ يفسو وَعِنْدنَا ... أَنه قد تبخرا) // مجزوء الْخَفِيف //
وَمن أُخْرَى
(أيلول والعيد واعتدال الْهَوَاء فِي اللَّيْل وَالنَّهَار ... )
(وَشهر شَوَّال فِي تكافي ... سَاعَات أَيَّامه الْقصار)
(أَرْبَعَة تقتضيك دين ... السماع وَاللَّهْو وَالْعَقار)
(فَاشْرَبْ لَهَا بالكبير ... إِن الْكَبِير للسادة الْكِبَار) // مخلع الْبَسِيط //
وَمن أُخْرَى
(والكأس تسلبني عَقْلِي وأهون مَا ... لهوت عَن ذكره عَقْلِي إِذا سلبا)
(حَمْرَاء يُمْسِي بناني وَهُوَ فَوق يَدي ... مِنْهَا بِمثل شُعَاع الشَّمْس مختضبا)
