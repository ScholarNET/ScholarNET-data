(والسجن عِنْدِي منزل التُّرَاب ... والهول عِنْدِي موقف الْحساب) // من الرجز //
وَله من أُخْرَى
(وَلَا تغترر بالحليم تغضبه ... فَرُبمَا أحرق الثرى الْبرد) // من المنسرح //
59 - أَبُو سعيد أَحْمد بن شبيب الشبيبي

فَرد خوارزم ومفخرتها وَكَانَ جَامعا بَين أدب الْقَلَم وَالسيف
وفروسية اللِّسَان والسنان صَاحب كتب وكتائب وفضائل ومناقب وَلما اخْتصَّ بالدولة السامانية
والدولة البويهية سمى صَاحب الجيشين وَشَيخ الدولتين وَقَالَ
(رب إِن ابْن شبيب أحمدا ... صَاحب الجيشين شيخ الدولتين)
(واثق بِاللَّه يَرْجُو الْمُصْطَفى ... وأخاه المرتضى والحسنين) // من الرمل //
وَسمعت أَبَا بكر الْخَوَارِزْمِيّ يَقُول كَانَ الشبيبي فِي أَيَّام شبابه بخوارزم يَقُول شعرًا غليظا جاسيا كأشعار المؤدبين فَلَمَّا عَاشر النَّاس وَلَقي الأفاضل لطف طبعه ورق شعره كَقَوْلِه وَكتب بِهِ إِلَى
(للشبيبي صنيعتك ... حسرات لفرقتك)
(واشتياق إِلَى لِقَاء ... تباشير طلعتك)
(رب سهل لقاءه ... يَا إلهي بِرَحْمَتك) // من مجزوء الْخَفِيف //
وأنشدني أَبُو عبد الله مُحَمَّد بن حَامِد قَالَ أَنْشدني أَبُو سعيد صَاحب الجيشين لنَفسِهِ فِي أبي بكر الْخَوَارِزْمِيّ
(أَبُو بكر لَهُ أدب وَفضل ... وَلَكِن لَا يَدُوم على الإخاء)
(مودته إِذا دَامَت لخل ... فَمن وَقت الصَّباح إِلَى الْمسَاء) // من الوافر //
وأنشدني غَيره لَهُ فِي الْأَمِير أبي نصر الميكالي
