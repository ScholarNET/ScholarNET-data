(فَلَمَّا رَأَيْت الْغدر من شَأْنك اغتدى ... غَدِير التصافي بَيْننَا متكدرا)
(فوَاللَّه مَا اهواك الا تكلفا ... وَلَا اشتكي الهجران الا تخمرا) // الطَّوِيل //
وَقَوله فِي انسان قصير ضئيل تزوج طَوِيلَة ضخمة من الْكَامِل
(يَا من احل بِهِ الرزيه ... واعاد نعْمَته بليه)
(حظي الردي بك اذ غَدَتْ ... لَك بنت عمار حظيه)
(قل لي وَكَيف تنيكها ... مَعَ دلّ قامتك القميه)
(انت الْبَعُوضَة قلَّة ... وَكَأَنَّهَا جمل الضحيه)
(نبئتها قَالَت وَقد ... بصرت بأيرك كالشظيه)
(من لَيْسَ تشبعه الهريسة ... كَيفَ تشبعه القليه)
(فَلَو اطَّلَعت عَلَيْهِمَا ... عِنْد ارتكابهما البليه)
(لذكرت فِي شخصيهما العنقاء ... قد خطفت صَبِيه) // الْكَامِل //
وَقَوله من الْخَفِيف
(قل لمن يَشْتَهِي المديح وَلَكِن ... دون معروفه مطال ولي)
(سَوف اهجوك بعد مدح وتحريك ... وعتب وَآخر الدَّاء كي) // الْخَفِيف //
وَقَوله من المنسرح
(بَغْدَاد قد صَار خَيرهَا شرا ... صيرها الله مثل سامرا)
(اطلب وفتش واحرص فلست ترى ... فِي اهلها حرَّة وَلَا حرا) // المنسرح //
وَقَوله من قصيدة من الْبَسِيط
(نيل المطالب بالهندية البتر ... لَا بالاماني والتأميل للقدر)
