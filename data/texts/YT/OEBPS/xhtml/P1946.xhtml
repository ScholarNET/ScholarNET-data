ملح وغرر من شعره

قَالَ
(لَهُ وَجه الْهلَال لنصف شهر ... وأجفان مكحلة بِسحر)
(فَعِنْدَ الابتسام كليل بدر ... وَعند الانتقام كَيَوْم بدر) // من الوافر //
وَقَالَ
(بنفسي من غَدا ضيفا عَزِيزًا ... عَليّ وَإِن لقِيت بِهِ عذَابا)
(ينَال هَوَاهُ من كَبِدِي كبابا ... وَيشْرب من دمي أبدا شرابًا) // من الوافر //
وَقَالَ
(أيا ضرَّة الشَّمْس المنيرة بالضحى ... وَمن عجزت عَن كنهها صفة الورى)
(عذرتك إِن لم أحظ مِنْك بِرُؤْيَة ... فَأَنت لعمري الرّوح وَالروح لَا ترى) // من الطَّوِيل //
وَقَالَ
(لي شادن مَا أُطِيق الدَّهْر هجرته ... أَمن يجرعني دَاء يداويني)
(شمس تظللني نجم يضللني ... مَاء يسكرني رَاح تصحيني) // من الْبَسِيط //
وَقَالَ
(إِنِّي أضن بحبيه على سقمي ... وَلَيْسَ وَالله دَاء الْحبّ بالأمم)
(قَالَ الطَّبِيب اقتصد يَوْمًا فَقلت لَهُ ... أخْشَى خُرُوج هَوَاهُ مَعَ خُرُوج دمي) // من الْبَسِيط //
وَقَالَ
