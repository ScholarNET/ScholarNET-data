(ابتعتها غير مغبون وَلَو طلب الْخمار ... روحي بهَا أَعْطَيْت مَا طلبا ... )
(وأربح النَّاس عِنْدِي فِي تِجَارَته ... مُحَصل يَشْتَرِي بِالْفِضَّةِ الذهبا) // الْبَسِيط //
وَمن أُخْرَى
(يَا صَاحِبي استيقظا من رقدة ... تزري عقل اللبيب الأكيس)
(هَذِه المجرة والنجوم كَأَنَّهَا ... نهر تدفق فِي حديقة نرجس)
(وَأرى الصِّبَا قد غسلت بنسيمها ... فعلام شربي الراح غير مغلس)
(قوما اسقياني قهوة رُومِية ... مذ عهد قَيْصر دنها لم يمس)
(صرفا تضيف إِذا تسلط حكمهَا ... موت الْعُقُول إِلَى حَيَاة الْأَنْفس) // الْكَامِل //
وَمن أُخْرَى
(من شُرُوط الصبوح فِي المهرجان ... خفَّة الشّغل من خلو الْمَكَان)
(وَحُضُور الطَّعَام قبل طُلُوع الشَّمْس ... مذ أمس بَارِد الألوان)
(والعروس الَّتِي تزف إِلَى ... الأرطال فِي ثوب صبغها الأرجواني)
(رسموا طين دنها وَهُوَ رطب ... باسم كسْرَى كسْرَى أنو شرْوَان)
(وَترى سوسن الكؤوس عَلَيْهَا ... كسْوَة من شقائق النُّعْمَان)
(ثمَّ خَفق الطبول بَين الأغاني ... واصطكاك الأوتار فِي العيدان)
(وَالسَّمَاع الَّذِي يمل على الأسماع مَا تشْتَهي بِلَا ترجمان ... )
(كل صَوت من اقتراحات إِسْحَاق ... الَّتِي زينت كتاب الأغاني)
(لَا أعد الصبوح إِلَّا غبوقا ... إِن جعلت الصبوح بعد الْأَذَان)
(يَا خليلي قد عطشت وَفِي ... الْخمْرَة ري للحائم العطشان)
