وَقَوله من الطَّوِيل
(أوفي كلا وقتي قسط تأله ... وقسط هوى لَا يسْتَمر لمحرم)
(وَلَذَّة وجدي من لذاذة مطربي ... أسر إِلَى نَفسِي وأعذب فِي فمي) // الطَّوِيل //
وَقَوله من الْكَامِل
(يَا عَارِفًا بالداء مطرح السُّؤَال عَن الدَّوَاء ... )
(الْعلم عِنْدِي كالغذاء ... فَهَل تعيش بِلَا غذَاء) // الْكَامِل //
وَقَوله من الرمل
(لَو توسطت إِذا لم تتْرك ... وكففت الْقلب عَن بعض الارب)
(كَانَ أَرْجَى لَك فِي العقبى من أَن ... تملا الدَّلْو إِلَى عقد الكرب) // الرمل //
وَقَوله من المتقارب
(هَب الْبَعْث لم يأتنا نَذره ... وجماحة النَّار لم تضرم)
(أَلَيْسَ بكاف لذِي فكرة ... حَيَاء الْمُسِيء من الْمُنعم) // المتقارب //
وَقَوله من الْكَامِل
(يَا من يسر بلذة الدُّنْيَا ... ويظنها خلقت لما يهوى)
(لَا تكذبن فَإِنَّهَا خلقت ... لينال زاهدها بهَا الاخرى) // الْكَامِل //
وَقَوله من الطَّوِيل
(بعثت الى رب البرايا رِسَالَة ... توسل لي مِنْهَا دُعَاء مناصح)
(فجَاء جوابي بالاجابة وانجلت ... بهَا كرب ضَاقَتْ بِهن الجوانح) // الطَّوِيل //
