وَقَالَ
(شيبي عَزِيز غير أَن شبيبتي ... علق كريم لَا يُجَاوِزهُ الأمل)
(من ذَا الَّذِي سَاوَى سَواد لحاظه ... ببياض عَيْنَيْهِ وحسبك ذَا الْمثل) // من الْكَامِل //
وَقَالَ
(تعلم من الأفعى أمالي طبعها ... وآنس إِذا أوحشت تعف عَن الذَّم)
(لَئِن كَانَ سم ناقع تَحت نابها ... فَفِي لَحمهَا ترياق غائلة السم) // من الطَّوِيل //
وَقَالَ
(يَا من يُقَابل ديناري بدرهمه ... أقصر فدعواك طَاوُوس بِلَا ريش)
(وَأي عيب لعين الشَّمْس إِن عميت ... أَو قصرت عَنهُ أبصار الخفافيش) // من الْبَسِيط //
وَقَالَ
(عَلَيْك بإغباب الْوِصَال فضده ... يُعِيد حبال الود مِنْك رثاثا)
(وَلَو كلف الْإِنْسَان رُؤْيَة وَجهه ... لطلقه بعد الثَّلَاث ثَلَاثًا) // من الطَّوِيل //
وَقَالَ
(أَظن زمَان السوء قارف أبنة ... فَإِنِّي أرَاهُ يتبع العلج والغمرا)
(زففت إِلَى دهري عروس كفايتي ... فَطلقهَا قبل الدُّخُول بهَا عشرا) // من الطَّوِيل //
وَقَالَ يعزي الشَّيْخ أَبَا الطّيب سهل بن أَحْمد بن سُلَيْمَان عَن ابْنه
(من مبلغ شيخ أهل الْعلم قاطبة ... عني رِسَالَة محزون وأواه)
(أولى البرايا بِحسن الصَّبْر ممتحنا ... من كل فتياه توقيع عَن الله) // من الْبَسِيط //
