(وهواه لَئِن تَأَخّر عني ... طول يومي إِنِّي سيحضر يومي) // الْخَفِيف //
وَقَالَ
(عَليّ كالغزال وكالغزالة ... رَأَيْت بِهِ هلالا فِي غلاله)
(كَأَن بَيَاض غرته رشاد ... كَأَن سَواد طرته ضلاله)
(كَأَن الله أرْسلهُ نَبيا ... وصير حسنه أقوى دلاله)
(إِذا مَا زِدْت وصلا زِدْت خبلا ... كَأَن حبال وصلك لي خباله) // الوافر //
وَقَالَ
(هَذَا عَليّ عَليّ فِي محاسنه ... كَأَنَّمَا وَصفه أَن يبلغ الأملا)
(وَكم أَقُول وَقد أَبْصرت طلعته ... هَذَا الَّذِي فِي طراز الله قد عملا) // الْبَسِيط //
وَقَالَ
(وشادن أصبح فَوق الصفه ... قد ظلم الصب وَمَا أنصفه)
(كم قلت إِذْ قبل كفي وَقد ... تيمني يَا لَيْت كفي شفه) // السَّرِيع //
وَقَالَ فِي مَعْنَاهُ
(أَبَا شُجَاع يَا شُجَاع الورى ... وَمن غَدا فِي حسنه قبله)
(قبل فمي إِن كنت لي مؤثرا ... فاليد لَا تعرف الْقبْلَة) // السَّرِيع //
وَقَالَ فِي مَعْنَاهُ
(وشادن جماله ... تقصر عَنهُ صِفَتي)
