وَقَالَ
(وَذي همة فِي حضيض الكنيف ... وقرنين فِي فلك المُشْتَرِي)
(دخلت عَلَيْهِ انتصاف النَّهَار ... على غَفلَة حِين لم يشْعر)
(وَبَين يَدَيْهِ رغيفان مَعَ ... سكرجة كَانَ فِيهَا مري)
(فَلَمَّا قعدت فسا فسوة ... فَلم تخط عصفتها منخري)
(وَأَقْبل يضرط فِي إثْرهَا ... فَقلت أقوم وَإِلَّا خري) // المتقارب //
وَقَالَ فِي شيخ بني بِعَجُوزٍ
(أفْصح وَدعنِي من الرموز ... قد دخل الشَّيْخ بالعجوز)
(من لي بهَا حِين ضاجعته ... فِي ذَلِك الْموضع الحريز)
(فَكنت أخرا على زليخا ... وَهِي إِلَى جَانب الْعَزِيز) // مخلع الْبَسِيط //
وَقَالَ وَقد ركب إِلَى قوم فَوجدَ بَعضهم نَائِما وَبَعْضهمْ شَارِب دَوَاء
(قد أَصْبحُوا كَمَا ترى ... مَا بَين نوم وخرا)
(قوم بَرِئت مِنْهُم ... لأَنهم مني برا)
(مَا إِن أرى مثلا لَهُم ... وَلَا أرى أَنِّي أرى) // مجزوء الرجز //
وَقَالَ وَقد عَاتب إنْسَانا على زلَّة فجَاء بأكبر مِنْهَا
(لي صديق جنى عَليّ ... مرَارًا فأكثرا)
(ثمَّ لما عتبته ... غسل الْبَوْل بالخرا) // مجزوء الْخَفِيف //
وَقَالَ
(فقدت بخْتِي إِنَّه ... مَا زَالَ بختا قذرا)
