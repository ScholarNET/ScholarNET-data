(قلت لطرف الطَّبْع لما ونى ... وَلم يطع أَمْرِي وَلَا زجري)
(مَالك لَا تجْرِي وَأَنت الَّذِي ... تحوي مدى الغايات إِذْ تجْرِي)
(فَقَالَ لي دَعْنِي وَلَا تؤذني ... حَتَّى مَتى أجري بِلَا أجر) // من السَّرِيع //
وَله
(للنَّاس فِي محن الزَّمَان مَرَاتِب ... ولكلهم فِيهَا نصيب راتب)
(وَكَأن أوفرهم إِذا استقريتهم ... مِنْهَا نَصِيبا شَاعِر أَو كَاتب)
(فَأَقل عتبك والعتاب مَعًا فَلم ... يسْعد بإعتاب الزَّمَان معاتب) // من الْكَامِل //
وَله
(جعلنَا أجنبيين ... بِلَا جرم وَلَا تبل)
(وأقصينا وَمَا خنا ... وَمَا زغنا عَن الْعدْل)
(فَقل لي يَا أَخا السؤدد ... والهمة وَالْفضل)
(إِلَى كم نَحن فِي ضيق ... وَفِي عزل وَفِي أزل)
(أما تنشط أَن تملي ... على الْكَاتِب أَنْتُم لي) // من الهزج //
وَله
(وجدت مَا قد بعثت غثا ... مستحقرا لَيْسَ بالثمين)
(فليت شعري قليت شعري ... فَكَانَ غثا بِلَا سمين) // من مخلع الْبَسِيط //
وَله
(إِذا ملك لم يكن ذاهبه ... فَدَعْهُ فدولته ذاهبه) // من المتقارب //
