(شعر لَو أَن الشهد ... قيس بِهِ وَجَدْنَاهُ زعاقا) // مجزوء الْكَامِل //
وَقَالَ يصف رمكة شقراء
(شقراء إِلَّا حجول مؤخرها ... فَهِيَ مدام ورسغها الزّبد)
(تعطيك مجهودها فراهتها ... فِي السّير فالحضر عِنْدهَا وتد) // المنسرح //
وَقَالَ
(قلت للنزلة حلي ... وانزلي غير لهاتي)
(واتركي حلقي بحقي ... فَهُوَ دهليز حَياتِي) // مجزوء الرمل //
وَقَالَ فِي غُلَام لَهُ كبر فَأخْرجهُ
(مَا تَرَكْنَاهُ وَفِيه ... لمحب من طباخ)
(هدر الطير وَمن ... عاداتنا أكل الْفِرَاخ) // مجزوء الرمل //
وَقَالَ
(وَهَامة نيطت بهَا لحية ... يظلم من قد قاسها باللحى)
(قد نصل الخضب إِلَى نصفهَا ... فَهِيَ كَمثل النَّمْل إِذْ أجنحا) // السَّرِيع //
وَقَالَ
(فَإِن كنت من هَاشم فِي الذرى ... فقد ينْبت الشوك وسط الأقاحي) // المتقارب //
وَقَالَ
(هُوَ الْبَحْر إِلَّا أَنه عذب مورد ... وَمن عجب أَن العذوبة فِي الْبَحْر) // الطَّوِيل //
