مَا اخْرُج من شعره فِي الْغَزل وَغَيره
فَمن ذَلِك قَوْله من الوافر
(أَرَانِي الله وَجهك كل يَوْم ... صباحا للتيمن وَالسُّرُور)
(وأمتع ناظري بصحيفتيه ... لأقرا الْحسن من تِلْكَ السطور) // الوافر //
وَقَوله من مجزوء الرمل
(يَا منى نَفسِي وَيَا حسبي ... من حسن وَطيب)
(سابقي بالوصل موتِي ... أَو مشيبي ومغيبي)
(فَهُوَ للفتيان فِي الدُّنْيَا ... بمرصاد قريب) // مجزوء الرمل //
وَله فِي غُلَام اسْمه غَرِيب من الوافر
(رعى الرَّحْمَن قوما ملكوني ... رشا قصر بلغت بِهِ المرادا)
(وسموه مَعَ الْقُرْبَى غَرِيبا ... كنور الْعين سموهُ سوادا) // الوافر //
وَقَوله من الْخَفِيف
(رب ليل قطعت فِيهِ خماري ... بغزال كَأَنَّهُ مخمور)
(ومصاد سرحت فِيهِ وَنصر ... بازيازي مظفر مَنْصُور)
(بصقور مثل النُّجُوم إِذا انْقَضتْ وعصف كأنهن صقور ... ) // الْخَفِيف //
