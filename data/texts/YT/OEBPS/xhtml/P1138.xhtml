6 - أَبُو الْحسن بن مقلة
من أَبنَاء الوزراء وَبَقِيَّة بني مقلة يَقُول
(لست ذَا ذلة إِذا عضني الدَّهْر ... وَلَا شامخا إِذا واتاني)
(أَنا نَار فِي مرتقى نفس الْحَاسِد ... مَاء جَار مَعَ الإخوان) // الْخَفِيف //
وَقَالَ من قصيدة
(وَإِذا رَأَيْت فَتى بِأَعْلَى رُتْبَة ... فِي شامخ من عزه المترفع)
(قَالَت لي النَّفس العروف بفضلها ... مَا كَانَ أولاني بِهَذَا الْموضع) // الْكَامِل //
وَقَالَ
(الدَّهْر يلْعَب بالفتى فيهيضه ... طورا وَيجْبر عظمه فيراش)
(وَكَذَا رَأينَا الدَّهْر فِي إعراضه ... ينحى وَفِي إقباله ينتاش) // الْكَامِل //
وَقَالَ
(أدل فيا حبذا من مدل ... وَمن ظَالِم لدمي مستحل)
(إِذا مَا تعزز قابلته ... بذل وَذَلِكَ جهد الْمقل) // المتقارب //
وَقَالَ
(أَنْت يَا ذَا الْخَال فِي الوجنة ... مِمَّا بِي خَال)
(لَا تبالي بِي وَلَا تخطرني ... مِنْك ببال)
(لَا وَلَا تفكر فِي حَالي ... وَقد تعرف حَالي)
(أَنا فِي النَّاس إمامي ... وَفِي حبك غالي) // الرمل //
