(هُوَ إِن جاد بلثم ... شفتاه شفتاه) // من مجزوء الرمل //

(صدف الحبيب بوصله ... فجفا رقادي إِذْ صدق)
(وَنَثَرت لُؤْلُؤ أدمع ... أضحى لَهَا جفني صدف) // من مجزوء الْكَامِل //
وَقَالَ
(مَاذَا عَلَيْهِ لَو أَبَاحَ رِيقه ... لقلب صب يشتكي حريقه) // من الرجز //
وَقَالَ
(بنفسي غزال صَار لِلْحسنِ كعبة ... يحجّ من الْفَج العميق ويعبد)
(دَعَاني الْهوى فِيهِ فلبيت طَائِعا ... وأحرمت بالإخلاص وَالسَّعْي يشْهد)
(فجفني للتسهيد والدمع قَارن ... وقلبي فِيهِ بالصبابة مُفْرد) // من الطَّوِيل //

قَالَ فِي الريحان
