(وصديق جَاءَنِي يسألني مَاذَا لديك ... )
(قلت عِنْدِي بَحر خمر ... حوله آجام نيك) // مجزوء الرمل //
وَقَوله من الطَّوِيل
(ولي صَاحب افسى الْبَريَّة كلهَا ... يشككني فِيهِ إِذا مَا تنفسا)
(تحولت الانفاس مِنْهُ الى استه ... فَمَا اُحْدُ يدْرِي تنفس ام فسا) // الطَّوِيل //
وَقَوله من مجزوء الْكَامِل
(لَيْسَ اشتقاق ابي المظفر ... من أَن يرى ظفرا فيظفر)
(لَكِن تطاول ظفره ... فلذاك قيل ابو المظفر) // مجزوء الْكَامِل //
