(أَو مَا رَأَيْت الْبَحْر ... يغرق مِنْهُ بالخطب الْيَسِير)
(وَالنَّاس مثل الْجِسْم يعْتَمد ... الْقَبِيل على الدبير)
(يتحامل الْعُضْو الخطير ... بِقُوَّة الْعُضْو الحقير)
(كتحامل الرمْح الطَّوِيل ... بزجه ذَاك الْقصير) // من مجزوء الْكَامِل //
وَمن أُخْرَى
(يَا أَيهَا الْخَاطِب مدحي وَهل ... يُورد من غير رشاء قليب)
(شَيْئَانِ لم يجتمعا لامرئ ... حب الدَّنَانِير وَحب الحبيب) // من السَّرِيع //
وَمن أُخْرَى
(ولي وَالله إخْوَان كثير ... نَصِيبي من فعالهم سَوَاء)
(وَلَكِنِّي رَأَيْتُك من أنَاس ... إِذا لم يحسنوا فَلَقَد أساءوا) // من الوافر //
وَمن أُخْرَى
(وَمَتى شتمت الدَّهْر تَشْتُم صَابِرًا ... تبْكي ويضحك ذَلِك المشتوم) // من الْكَامِل //
لَا وَمن صاحبية لما ورد حَضرته مَكْتُوب من جِهَة تاش
(فَإِن ردني دهري عَلَيْك طريدة ... فَلَا غرو أَن يسترجع الْقوس حَاجِب)
(هُوَ الوكر طرنا والريش وَافد ... وعدنا إِلَيْهِ الْآن والريش ذَاهِب) // من الطَّوِيل //
وَمِنْهَا
(جزى الله عني أهل سامان مَا أَتَوا ... وَفِي الله للثأر المضيع طَالب)
(هم زوجوني الْهم بعد طَلَاقه ... وَذَلِكَ عرس للمآتم جالب)
