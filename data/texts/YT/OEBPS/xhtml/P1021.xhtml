وَقَالَ
(رب عَجُوز مستعينيه ... سلقية اللَّوْن سلوقيه)
(عاجية الشّعْر إِذا استضحكت ... أبدت ثنايا آبنوسيه)
(ذَات حر عنبله بارز ... كمرقب فِي وسط بريه)
(وشعرة بالقمل منظومة ... كالودع فِي عقصة كرديه)
(يفتر ذَاك الصدغ عَن بظرها ... كقنفذ عض على ريه)
(مُسِنَّة تصبو إِلَى أَمْرَد ... فَهِيَ على العاهة لوطيه) // السَّرِيع //
وَقَالَ
(عجبت لخمرة البخراء أَنِّي ... أَقَامَت مَعَ مؤاجرها زَمَانا)
(وَلَيْسَ لأيره طول وَلَكِن ... ينيك بِهِ فيردفه لِسَانا)
(لحاه الله كَيفَ يدس فِيهَا ... لِسَانا رُبمَا درس القرانا) // الوافر //
وَقَالَ
(هَل لَك يَا خمرة فِي تجرة ... مربحة مَا مثلهَا تجره)
(صيري إِلَى الْبَصْرَة واسترزقي ... رَبك بالنكهة فِي البصره)
(فَلَو عرضت الرِّيق فِي سوقها ... لابتيعت التفلة بالبدره)
(تزكو بهَا النّخل وتحمر فِي ... غير أَوَان الْحمرَة البسره) // السَّرِيع //
