وَقَوله من الْكَامِل
(دع قَول واشية وواشي ... واجعلهما كَلْبِي هراش)
(واشرب مُعتقة تسلسل ... فِي الْعِظَام وَفِي المحاشي)
(حَتَّى ترى الْعود المسن ... بهَا أرق من الخشاش) // الْكَامِل //
وَقَوله من الْكَامِل
(ألحاظ عين تَنْتَهِي ... فِي روض ورد تزدهي)
(رتعت بهَا وتنزهت ... مِنْهَا بِأَيّ تنزه)
(يَا أَيهَا الخنث الجفون ... بنخوة وَتكره)
(والمكتفي عجبا أما ... ترثي لأشعث أمره) // الْكَامِل //
وَقَوله من الْكَامِل
(أطفت شرارة لهوى ... ولوت بشرة عدوي)
(شعل علون مفارقي ... وَمَضَت ببهجة سروي)
(لما شَككت عروضها ... ذهب الزحاف بحزوي)
(يَا أَيهَا الشادي صه ... لَيست بساعة شدو) // الْكَامِل //
وَقَوله من الهزج
(أَلا يَا زين قلبِي للشباب ... العفر إِذْ ولى)
