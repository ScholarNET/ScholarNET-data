(من يكن يهواه لِلْخلقِ فَإِنِّي عبد خلقه ... )
(إِن حسن الْخلق أبهى ... للفتى من حسن خلقه) // مجزوء الرمل //
وَقَوله من الْبَسِيط
(قَالُوا عشقت صَغِيرا قلت أرتع فِي ... روض المحاسن حَتَّى يدْرك الثَّمر)
(ربيع حسن دَعَاني لافتتاح هوى ... لما تفتح مِنْهُ النُّور والزهر) // الْبَسِيط //
وَقَوله من المنسرح
(وددت اني بكفه قلم ... أَو انني مُدَّة على قلمه)
(يأخذني مرّة ويلثمني ... إِن علقت مِنْهُ شَعْرَة بفمه) // المنسرح //
وَقَوله من الْبَسِيط
(قد قلت إِذْ خَان صبري من كلفت بِهِ ... وَلم يكن عَنهُ لي صَبر وَلَا جلد)
(إِن كَانَ شاركني فِي حبه وقح ... فالنهر يشرب مِنْهُ الْكَلْب والاسد) // الْبَسِيط //
وَقَوله من الْكَامِل
(لَا تعشقن ابْن الرّبيع فَإِنَّهُ ... عِنْد التجرد آيَة الايات)
(وَجه كعبادان لَيْسَ وَرَاءه ... لمحبه شَيْء سوى الخشبات) // الْكَامِل //
