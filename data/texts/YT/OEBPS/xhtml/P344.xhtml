(لم يصلني وَلم يعدني وَقَالَ اكتم ... فَمَاذَا أسر حَتَّى أكاتم) // من الْخَفِيف //
وَقَوله
(قبلتها أشتفي بقبلتها ... فزادني ذَلِك اللمى ألما)
(وساءلتني عَن مبتدا سقمي ... مسقم جفنيك مسقمي بهما) // من المنسرح //
وَقَوله
(يَا عِلّة الأجفان كفي كفي ... مَا حملت مِنْك وَمَا استوثقت)
(وساعدينا واعلمي أَنَّهَا ... قد نذرت قَتْلِي وَمَا أعتقت) // من السَّرِيع //
وَقَوله
(أرى اللَّيَالِي إِذا عاتبتها جعلت ... تمن أَن جَعَلتني من ذَوي الْأَدَب)
(وَلَيْسَ عِنْد اللَّيَالِي أَن أقبح مَا ... صنعن بِي أَن جعلن الشّعْر مكتسبي)
(إِن كَانَ لَا بُد من مدح فها أَنا ذَا ... بِحَيْثُ آمن فِي قولي من الْكَذِب) // من الْبَسِيط //
وَقَوله
(إِذا كسدت سوق الثَّنَاء فجوده ... طلوب لأسباب الثَّنَاء كسوب)
(تضيق بِمَا تحوي يَدَاهُ وصدره ... بتفريق مَا تحوي يَدَاهُ رحيب) // من الطَّوِيل //
وَقَوله
(وغزال مثل الغزالة يحكيها ... كمالا إِلَّا بقلب وود)
(رق جسما فرق دمعي عَلَيْهِ ... فَجرى مثل خَدّه فَوق خدي) // من الْخَفِيف //
وَقَوله
(وَالله مَا عورضت فِي مهجتي ... إِلَّا لِأَن أرفع عَنْهَا يَدي)
