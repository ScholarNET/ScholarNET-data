(فَابْعَثْ إِلَيّ مَعَ الرَّسُول ... إِذا أَتَاك بملء دن)
(وَمَتى رضيت بِأَن أقطع ... أَو أهجن أَو أزني)
(فاصرف رَسُولي خائبا ... وادفع بقبحك حسن ظَنِّي) // مجزوء الْكَامِل //
وَقَالَ
(يَا فَتى الْجَصَّاص قد أعدمتني ... الْإِحْسَان دَفعه)
(ولزمت الشُّح بِالرَّاحِ ... فَمَا تسخو بجرعه)
(قد أَتَى الْعِيد وصحوي ... فِيهِ يَا مولَايَ بدعه)
(أملي فِيك قريب ... لَيْسَ فِيهِ لي مَنعه)
(شربة من خمرك الصافي ... فِي وَمن ندك قطعه)
(ينْبذ الْحبّ فيستنفده ... الشّعْر برقعه) // مجزوء الرمل //
وَقَالَ
(لنا على النَّار قدر ... بِخَاتم النَّار بكر)
(وَعِنْدنَا من بقايا ... صَبِيحَة الْعِيد خمر)
(وَقد دَعونَا غُلَاما ... كالغصن أَعْلَاهُ بدر)
(فَاطلع علينا وساعد ... أَو لَا فَمَا لَك عذر) // المجتث //
وَقَالَ
(على الأثافي لنا قدرو ... سَاكِنة النبض لَا تَفُور)
(قَامَت على سوقها لأكل ... وَنحن من حولهَا ندور)
(وَعِنْدنَا من شراب عَمْرو ... دن رحيب الحشى كَبِير)
