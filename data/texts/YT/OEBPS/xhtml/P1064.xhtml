(قد وَقع الصُّلْح على غلتي ... فاقتسموها كارة كَارِه)
(لَا يدبر الْبَقَّال إِلَّا إِذا ... تصالح السنور والفاره) // السَّرِيع //

(أَيهَا السَّائِل عَن حَالي ... أَنا الْمَضْرُوب زيد)
(وَأَنا الْمَحْبُوس لَكِن ... لَيْسَ فِي رجْلي قيد) // الرمل //
وَقَالَ
(وَقَائِل هُوَ رَأس الْعمَّال ... بَين النَّاس)

(هَذَا هُوَ الْحق وَالْحق ... مَا بِهِ من باس) // المجتث //
