(كَأَن سَواد اللَّيْل وَالْفَجْر ضَاحِك ... يلوح ويخفي اسود يتبسم) // الطَّوِيل //
وَقَالَ فِي غور الْكَوَاكِب عِنْد الصَّباح من الْبَسِيط
(عهدي بهَا وضياء الصُّبْح يطفئها ... كالسرج تطفأ أَو كالاعين العور)
(اعْجَبْ بِهِ حِين وافى وَهِي نيرة ... فظل يطمس مِنْهَا النُّور بِالنورِ) // الْبَسِيط //
وَقَالَ من سَائِر الاوصاف والتشبيهات من مجزوء الرمل
(بَات يسقيني وَيشْرب ... ذَهَبا للهم مَذْهَب)
(شادن يحمل مَاء ... فِيهِ نَار تتلهب)
(وردة ضاحكة عَن ... اقحوان حِين يقطب)
(لَو أدرناها على ميت لَكَانَ الْمَيِّت يطرب ... )
(لَيْت شعري اسرورا ... أم مداما بت أشْرب)
(صب فِي الكاسات مِنْهَا ... كالشهاب المتصوب)
(فَرَأَيْت الراح شرقا ... وَرَأَيْت الْهم مغرب)
(غُصْن فَوق كثيب ... ونهار تَحت غيهب)
(لَك مِنْهُ مطرب يرضيك إِن شِئْت ومضرب ... )
(جنَّة عذبت فِيهَا ... بتجن وتجنب)
(هَل رَأَيْتُمْ أحدا قبلي بِالْجنَّةِ عذب ... )
(بِأبي أَنْت وَأمي ... من بعيد حِين تقرب)
(لي قلب كَيفَ مَا قلبه الله يقلب ... )
(وجفون يغْضب الغمض عَلَيْهَا حِين يغْضب ... )
(رب ليل كتجنيك مُقيم لَيْسَ يذهب ... )
