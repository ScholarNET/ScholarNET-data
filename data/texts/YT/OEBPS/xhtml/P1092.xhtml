(قيل لأيري وَقد رَأَوْهُ وَلَا ... الهارب بعد الْحُصُول فِي الْأسر)
(يشْتَد بعد العشا إِلَى حرهَا ... عدوا بِلَا حشمة وَلَا فكر)
(مَا لَك هوذا تطير قَالَ لَهُم ... أطير مستعجلا إِلَى وَكري)
(ولي خصى لَو خرجت أعرضه اشْتَرَاهُ ... مني بِرُوحِهِ دري)
(ايري عَلَيْهِ كَأَنَّهُ وتد ... قد علقت فِيهِ دبة البزر) // المنسرح //

(يَا وَيحكم وَاللَّحم يعرض ... والبزاة على الكنادر)
(قومُوا بِنَا نحشو البظور ... بفيشنا حَشْو الْمسَاوِر)
(نبدا بكراعاتهم ... ونعود نعثر بالزوامر)
(ثمَّ الحوافظ إنَّهُنَّ ... عَجَائِز شمط عواهر)
(أحراحهم بيض العنافق ... واللحى سود المباعر)
(كشيوخ أَصْحَاب الحديثة إِذا تَمْشُوا بالمحابر) // مجزوء الْكَامِل //

(حبائب مثل حَصى عكبرا ... والرقبا مثل نوى البصره)
