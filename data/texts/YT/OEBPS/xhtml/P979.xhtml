وَقَالَ من المتقارب
(عذارك جَادَتْ عَلَيْهِ الرياض ... بأجفانها وبآماقها)
(وَطَالَ غرام الغواني بِهِ ... فقد طرزته بأحداقها) // المتقارب //
وَقَالَ من الْخَفِيف
(فاض مَاء الْجمال فِي الاقطار ... كل بدر مطرز بعذار)
(قد ارانا عقارب الشّعْر من خديه ... تأوي مكامن الجلنار) // الْخَفِيف //
وَقَالَ من قصيدة من المتقارب
(يفض الغزال جفون الْغَزل ... وَقد فَضَح فِيهَا الْكحل)
(وَلَا وجني الْورْد فِي وجنتيه مَا أوجب اللثم ذَاك الخجل ... ) // المتقارب //
وَقَالَ من اخرى من الْكَامِل
(مَا تسرع الالحاظ تخطف وردة ... من خَدّه إِلَّا عثرن بخاله)
(مذ نقبوه وزرفنوا أصداغه ... ختموا بغالية على أقفاله) // الْكَامِل //
وَقَالَ من الرجز
(تعرض الشّعْر لعارضيه ... وَأطلق العشاق من يَدَيْهِ)
(كَأَن الصِّبَا يَهْتَز فِي عطفيه ... وَالْحسن تجْرِي خيله إِلَيْهِ)
(حَتَّى إِذا ابصر وجنتيه ... حجبتا بِمثل حاجبيه)
(جاد عذاريه بعبرتيه ... كَأَنَّمَا يغسل من خديه)
(صحيفَة قد كتبت عَلَيْهِ ... ) // الرجز //
