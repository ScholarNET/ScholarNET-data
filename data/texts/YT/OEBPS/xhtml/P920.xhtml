وَقَالَ ايضا من الْخَفِيف
(قد شربنا عل شقائق روض ... شربت عِبْرَة السَّحَاب السكوب)
(صبغت من دم الْقُلُوب فَمَا تبصر إِلَّا تعلّقت بالقلوب ... ) // الْخَفِيف //
وَقَالَ ايضا من المنسرح
(امْر غَد انت مِنْهُ فِي لبس ... وأمس قد فَاتَ فاله عَن امس)
(وَإِنَّمَا الْعَيْش عَيْش وقتك ذَا ... فبادر الشَّمْس بابنة الشَّمْس) // المنسرح //
وَقَالَ ايضا من الوافر
(اقول لصاحبي والراح روح ... لجسم الكأس فِي كف النديم)
(وَقد حبس الدجى عَنَّا بواك ... تسيل نفوسها فَوق الجسوم)
(وَنحن من المسرة فِي سَمَاء ... فَمن سارى الضياء وَمن مُقيم)
(شموعك والكؤوس مَعَ الندامى ... نُجُوم فِي نُجُوم فِي نُجُوم) // الوافر //
وَقَالَ فِي قلَّة شربه وَسُرْعَة سكره من الوافر
(فديتك لَو علمت بِبَعْض مَا بِي ... لما جرعتني إِلَّا بمسعط)
(فحسبك أَن كرما فِي جواري ... أَمر بِبَابِهِ فأكاد أسقط) // الوافر //
وَله فِي مثل ذَلِك من المجتث
(لَو انني مسعي ... شربت مَا شِئْت حينا)
(لكنني عهدي ... فاعرف حَدِيثي يَقِينا)
(قَرَأت عُهْدَة كرم ... فَكَانَ سكري سنينا) // المجتث //
