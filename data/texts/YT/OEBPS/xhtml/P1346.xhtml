(قَالَ هَيْهَات أَنْت والنحس تربان ... وَقد كنتما رضيعي لبان)
(لَا تؤمل ركُوب متن سوى النعش وَلَا خلعة سوى الأكفان ... ) // الْخَفِيف //
وَله من أَبْيَات
(تكلفني التصبر والتسلي ... وَهل يسطاع إِلَّا المستطاع)
(وَقَالُوا قسْمَة نزلت بِعدْل ... فَقُلْنَا ليته جور مشَاع) // الوافر //

(تعيب الغانيات عَليّ شيبي ... وتخفي شيبها عني المقانع)
(وَقَالَ لي العذول تعز عَنْهَا ... وَإِلَّا فانظرن مَا أَنْت صانع)
(فَقلت لَهُ مَتى قدمت خيرا ... وأيرا بعده لَيست تمانع) // الوافر //
