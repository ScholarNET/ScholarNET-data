(تسومني الصَّبْر عَن روحي وتمنعني ... عَن ذكره إِن ذَا من رَأْيك الْغَلَط) // الْبَسِيط //
وَقَوله من الوافر
(ترى العشاق لاقوا مَا أُلَاقِي ... فقد بلغت بِي النَّفس التراقي)
(خصصت من الْهوى بِأَمْر شَيْء ... وَكنت ارى الْهوى عذب المذاق)
(أَنا العَبْد الَّذِي لَا عتق يَرْجُو ... وَلَا يجد السَّبِيل إِلَى الاباق) // الوافر //
وَقَوله من الطَّوِيل
(وَمَا سرني أَن الْهوى غير صَاحِبي ... وَأَن خراج العبشميين فِي ملكي)
(وَلَا كنت أرْضى أَن أرى متخليا ... من الْحبّ لَو أعْطى بِهِ خَاتم الْملك)
(نسيم الْهوى أذكى وَإِن جَار واعتدى ... على أنف العشاق من نفحة الْمسك) // الطَّوِيل //
وَقَوله من الطَّوِيل
(وَمن يحمد الصَّبْر الْجَمِيل على الْهوى ... فَإِن خلاف الصَّبْر عِنْدِي احْمَد)
(إِذا كَانَ قلب الْمَرْء لَا يألم النَّوَى ... ويشكو لظى نيرانها فَهُوَ جلمد) // الطَّوِيل //
وَقَوله من الْكَامِل
(احوى النواظر ألعس الشفتين عذب الرِّيق آلمى ... )
(مخضر شَاربه علا ... درا يُرِيك الدّرّ نظما)
(لَو زارني طيف لَهُ ... عِنْد الهجوع وَلَو ألما)
(لاعاد روحا أَو لفرج ... من هموم النَّفس هما) // الْكَامِل //
